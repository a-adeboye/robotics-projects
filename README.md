# Robotics Projects 📡

This repository contains hands-on robotics projects and exercises created while following the book **"Learning Robotics using Python" by Lentin**.  
It documents my **progress, code experiments, and personal notes** as I learn robotics programming, simulation, and control.

---

## 🧩 Purpose

- Track learning from *Learning Robotics using Python*  
- Implement Python-based robotics exercises and simulations  
- Experiment with sensors, actuators, and control systems  
- Build a foundation for more advanced robotics and intelligent control projects

---

## 🛠 Tools & Environment

- **Programming Language:** Python 3.x  
- **Libraries:** `numpy`, `matplotlib`, `rospy`, `pygame`, `pybullet` (where applicable)  
- **Development Platform:** Linux / Azure Educate instance / Termux for remote management  
- **Version Control:** Git & GitHub  

---

## 🚀 Learning Goals

- Understand **robot kinematics and dynamics**  
- Implement **basic sensors and actuators in Python**  
- Learn to control robots using **software simulations**  
- Build skills in **Python-based robotics frameworks** (e.g., ROS)
